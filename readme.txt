== Description ==

Grow you email list which you can use for marketing your products by offering your visitors a chance to win coupon, discount, product, etc. through spinning the wheel of fortune.
The most optimal way of funny email collection ever found.

* Beautiful different templates.
* Export emails directly into MailChimp list, ActiveCampaign, etc.
* Create unlimited wheels, use them for pages and as widget.
* Fully configure the chance of winning.
* Configure quantity of wheel size, each wining and lose item
* Control when to show the popup.
* Compatible in any language(WPML).
* Complete control over the content (text, buttons, ...) displayed in the popup.
* Customizable email templates.
* And more ...

== Installation ==

Installing the plugin is very easy:

1. Install from the WordPress Dashboard > Plugin page
2. Or manually download the plugin and extract to wp-content/plugins/
3. Activate the plugin
4. Go to Wordpress Dashboard > Settings > WP Spinner
5. Create your wheels
